小丑牌 筹码计算

 

此网页可以帮助你记下 小丑牌 (Balatro) 中，你的小丑牌对筹码的增益能让你到达哪一个关卡。


牌型基本数值 筹码 X 倍率

在此处填写你要打出的牌型的基本筹码和倍率。


出牌次数

在此处填写你可以出牌的次数。


平均牌筹码

在此处填写你打出的有效牌 (参与计分的牌) 的筹码和的期望数值。


小丑牌加成

填写内容含义
+50c+50 筹码
+10m+10 倍率
x1.5mX1.5 倍率
,使用逗号分隔

在此处按照顺序填写你的小丑牌能给你的牌型多少加成。


选择赌注颜色: 白注, 红注 || 绿注, 黑注, 蓝注 || 紫注及以上



最大总分期望:

颜色含义
蓝色一次出牌即可通关
深绿色总分数 >= 要求分数 * 2
绿色总分数 >= 要求分数 * 1.5
黄色总分数 >= 要求分数 * 1.1
橙色总分数 >= 要求分数 * 0.9
红色总分数 >= 要求分数 * 0.75
深红色总分数 < 要求分数 * 0.75
