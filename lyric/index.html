如何制作 XTSGAMES Lyric 文件

1. 新建一个 JSON 文件

2. 在根对象中创建以下键: "signature", "version", "musicInfo", "settings", "lyrics"

其中 "signature" 的值一定为 "xtsgames-lyric", "version" 的值为目前版本

3. "musicInfo" 的值为对象, 其中包括 "title" (字符串), "author" (字符串), "bpm" (数字), 分别填写歌曲的 标题，作者，每分钟节拍数

4. "settings" 的值为对象, 其中包括可选键 (不存在即使用默认设置) "showTitle", "showDot", "showCircle" (布尔值) 和 颜色 (格式为 "#abcdef"): "backgroundColor", "lyricColor", "activeColor" (字符串)

5. "lyrics" 的值为数组，该数组中的每一个数组就是一句歌词，一句歌词数组由若干个以下数组构成: ["汉字(或单词音节)", 时长(以节拍为单位)] (一共有三层数组)

如果是休息段 (伴奏), 在汉字的位置输入 [SPACE] ([SPACE] 必须独立成句)

你可以使用 节拍量化器 来更方便地获取每个音符的长度
